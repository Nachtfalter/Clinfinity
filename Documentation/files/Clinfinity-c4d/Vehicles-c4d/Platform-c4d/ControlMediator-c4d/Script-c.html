Master/slave system

Methods that concern the tree-like structure of connected control mediators.  Each mediator can have a left and a right slave mediator, it then acts as the master to these slaves.  Likewise, the slave mediators can have slaves themselves.  This way, an entire tree of mediators can be built.

Each control event and movement event that is fired in a slave gets routed to its master.  This is repeated until the event reaches the “absolute” master at the top of the tree.  From there, the event gets routed back down in the tree, and additionally, to all listeners for the respective event.  This way, any event fired anywhere in the tree can be received by any listener in the tree.
